Install autofs and cifs utils 

apt-get install autofs cifs-utils
In the /etc/auto.master at the end add 

/opt/ /etc/auto.cifs --timeout=60 --ghost
Then create file /etc/auto.cifs with 600 and following content 

 -fstype=cifs,rw,username=,password=,uid=,gid= :///
In this file, list more share sub folders if you have them. To test automount, use sudo automount -f -v Then use systemd to auto run it 

sudo systemctl enable autofs
sudo systemctl start autofs
